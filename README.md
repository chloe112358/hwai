## 個人數位名片（Notion 風格 + 河狸驚喜）

### 本機開啟
- 直接用瀏覽器開 `index.html` 即可。

### 取得「任何人都能開啟」的連結（建議：GitHub Pages）
1. 把整個資料夾內容上傳到 GitHub 的一個新 repo（例如 `my-card`）。
2. 到 GitHub repo 的 **Settings → Pages**
3. **Build and deployment**
   - Source 選 **Deploy from a branch**
   - Branch 選 `main`（或 `master`）/ root
4. 等待 1–2 分鐘，GitHub 會給你一個公開網址，任何人都能開啟（手機也可以）。
